# Hotel Booking Confirmation Page

This project is a Next.js starter built with TypeScript, Tailwind CSS, and a structured src folder for app, components, features, services, hooks, store, lib, types, and constants.

## Getting Started

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open http://localhost:3000 to view the app.

## Project Structure

- src/app for route groups and pages
- src/components for shared UI
- src/features for feature-based modules
- src/services, src/store, src/lib, src/types, and src/constants for supporting app layers
